 API-эндпоинты:

- http://127.0.0.1:8000/api/v1/task/ - для создание новой задачи и получение списка всех задач

- http://127.0.0.1:8000/api/v1/task/1/ - для получение задачи по идентификатору, обновление и удаление задачи
